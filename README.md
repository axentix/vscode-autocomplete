# Axentix VSCode autocomplete

A vscode extension that allows you to get axentix classes autocomplete.

<br>

## Enable extension

```
Ctrl + Shift + P
```
```
Axentix autocomplete
```

<br>

## Disable extension

```
Ctrl + Shift + P
```
```
Reload window
```
